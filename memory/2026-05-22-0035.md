### Fixed encoding bug (root cause!)
- **File:** `adapter-utils/dist/server-utils.js` linhas 1532 e 1548
- **Bug:** `String(chunk)` no callback `data` do stdout/stderr spawn child process
- No Windows, `String(buffer)` usa code page 850 (sistema), nao UTF-8
- Isso corrompia todos os caracteres acentuados (ç, ã, á, ê, etc.) → viravam `?`
- **Fix:** Mudei para `typeof chunk === "string" ? chunk : chunk.toString("utf8")` em ambos stdout e stderr
- Afeta TODOS os agents `opencode_local` no Windows
