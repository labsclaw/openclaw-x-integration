## Regra: Eng. Percepção deve usar Nemotron "omni" sempre

O engenheiro de percepção (Eng. Percepção) **deve usar o modelo Nemotron "omni" (nvidia/nemotron-3-nano-omni-30b-a3b-reasoning:free) sempre**, não o reasoning.

- "Omni" = non-reasoning → mais rápido e barato
- Não usar variantes "reasoning" para tarefas de visão computacional
- Reason: o reasoning é mais lento e mais caro, sem ganho significativo em análise visual
- Fallback se omni estiver bloqueado: aguardar instrução, não trocar automaticamente para outro modelo
